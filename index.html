Welcome to a public UDD mirror

This is a public, unofficial mirror of the Ultimate Debian Database.

How to use it

Use these credentials:

  • Host: public-udd-mirror.xvm.mit.edu
  • Port: 5432
  • Database: udd
  • Username: public-udd-mirror
  • Password: public-udd-mirror

Or (remember, the password is public-udd-mirror):

psql --host=public-udd-mirror.xvm.mit.edu --user=public-udd-mirror udd --password

It's also available on port 465 for the convenience of people behind weird firewalls.

Help

The best way to report issues is using Github issues.

I will also track to-do entries there.

The debian-qa mailing list is the best way to reach the UDD team. You can also find Asheesh there, and he doesn't mind if you discuss problems on the mailing list instead of via personal mail to him. Note that since this mirror is unofficial, it is provided on a best-effort basis and Asheesh is fundamentally responsible for maintaining server uptime, not anyone else on the debian-qa list.
